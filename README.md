# Student Helper
Student Helper - это веб-приложение, разработанное для того, чтобы помочь студентам эффективно организовывать свои академические задания и сроки их выполнения. Этот проект, созданный с использованием платформы Flask на Python, предоставляет удобный интерфейс для управления предметами, задачами и сроками, обеспечивая при этом безопасность данных и масштабируемость.  
# Основные функции:  
✅ Регистрация и вход в систему  
📁 Добавление учебных предметов  
📝 Создание задач с дедлайном  
🟢 Отметка задач как выполненных  
🗑 Удаление задач  
📊 Статистика: количество и % выполненных и невыполненных задач  
⏳ Показ ближайших 5 невыполненных дедлайнов на главной странице  
# Используемые технологии:  
Серверная часть: Flask (Python)  
База данных: PostgreSQL  
ORM: SQLAlchemy  
Интерфейс: HTML, CSS, JavaScript  
Аутентификация: Werkzeug (хэширование паролей)  
Разработка API: принципы RESTful
